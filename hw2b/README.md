
### Выделенные области и их параметры

**1. Bounding Box фильтрация**
Выделен объемный куб в пространстве с координатами:
- X ∈ [20, 50]
- Y ∈ [30, 70]
- Z ∈ [10, 40]


Эта область представляет центральную часть исходного куба, удаляя точки, расположенные близко к границам сцены.

**2. Фильтрация по высоте**
Выделены все точки с координатой Z > 80. Это соответствует верхним 20% объема исходного куба, что позволяет изолировать "верхние" поверхности для анализа.

**3. Фильтрация по расстоянию до центра**
Оставлены точки, находящиеся в радиусе 20 единиц от центральной точки (50, 50, 50). Формируется сферическая область в середине облака.

**4. Дополнительная фильтрация по кругу в плоскости XY**
Выделены точки внутри круга радиусом 15 единиц в горизонтальной плоскости с центром (50, 50), без ограничений по высоте.

### Количество точек до и после фильтрации

- **Исходное облако**: 100,000 точек
- **После BBox фильтрации**: 3,482 точки (3.48% от исходного)
- **После фильтрации по высоте**: 20,150 точек (20.15% от исходного)  
- **После фильтрации по расстоянию**: 3,296 точек (3.30% от исходного)
- **После фильтрации по кругу XY**: 6,956 точек (6.96% от исходного)

### Выводы по сравнению результатов

1. **Эффективность фильтрации** существенно различается: BBox и фильтр по расстоянию оставляют лишь ~3% точек, тогда как фильтр по высоте сохраняет 20% данных.

2. **Геометрия выделенных областей** влияет на количество точек: сферическая область (радиус 20) содержит примерно столько же точек, сколько прямоугольный BBox, хотя их объемы различаются.

3. **Фильтр по кругу в плоскости XY** занимает промежуточное положение, удаляя точки по горизонтальному расстоянию, но сохраняя весь вертикальный диапазон.

4. **Практическое применение**: BBox идеален для выделения объектов с четкими границами, фильтр по высоте — для анализа рельефа или поверхностей, фильтр по расстоянию — для изоляции сферических объектов или зон влияния.
