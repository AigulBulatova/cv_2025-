## Вопрос 3: Преимущества PointNet перед 3D CNN
1. Прямая обработка точек - не требует вокселизации или проекций, сохраняет всю геометрическую информацию.

2. Вычислительная эффективность - в 8-140 раз меньше операций по сравнению с 3D CNN и Multi-View CNN.

3. Линейная масштабируемость - сложность O(N) позволяет обрабатывать миллионы точек.

4. Встроенная инвариантность - T-Net автоматически выравнивает входные данные без augmentation.

5. Устойчивость к неполным данным - потеря лишь 2-4% точности при 50% отсутствующих точек.

6. Интерпретируемость - можно визуализировать critical points и понять, какие точки влияют на решение.

7. Универсальность - одна архитектура для классификации, сегментации и детекции.
