## Ответы на контрольные вопросы

### 1. Почему для RGB значения используются числа от 0 до 1 или от 0 до 255?
Диапазон 0-1 используется для нормализованных значений в вычислениях, где 0 соответствует отсутствию цвета, а 1 — максимальной интенсивности. Диапазон 0-255 соответствует 8-битному представлению, которое является стандартом для большинства дисплеев и графических форматов.

### 2. Как можно задать цвет точек на основе скалярной величины?
Сначала нужно нормализовать скалярные значения в диапазон 0-1, затем применить цветовую карту (colormap) для преобразования каждого значения в соответствующий RGB-цвет. В Matplotlib это делается через параметр `c=` с указанием cmap.

### 3. Чем отличается статичная визуализация (Matplotlib) от интерактивной (Plotly)?
Matplotlib создает статические изображения, подходящие для печати и вставки в документы. Plotly генерирует интерактивные веб-визуализации с возможностью вращения, масштабирования, выделения областей и просмотра значений точек при наведении курсора.

### 4. Можно ли использовать другую цветовую карту (colormap) в Matplotlib? Как?
Да, Matplotlib предоставляет множество цветовых карт. Для использования нужно указать параметр `cmap='название_карты'` в функциях визуализации. Доступные карты можно посмотреть через `plt.colormaps()`.

### 5. Что произойдёт, если не нормализовать значения при генерации цветов?
Если значения выходят за пределы диапазона 0-1 (для float) или 0-255 (для integer), они будут автоматически обрезаны до границ диапазона. Это может привести к потере градиента, некорректному отображению цветов или визуальным артефактам.
