## Задание 1

В качестве исходных данных использовалось облако из 50237 точек.

### Таблица с результатами

| Метод |  Точек после |  Время | Параметры |
|-------|-------------|------------|-------|-----------|
| Random |  25036 |  0.194 сек | 50% точек |
| Spatial |  8817 |  0.279 сек | 50 |
| Octree |  25701 |  0.181 сек | level = 7 |

**1. Какой метод дал наилучший визуальный результат при уменьшении количества точек вдвое?**

Octree показал лучший визуальный результат, при уменьшении на 56% форма сохранилась равномерно, без явных пробелов. Random при 50% тоже неплох, но распределение точек менее равномерное.

**2. Как изменяется точность геометрии при различных методах?**

Random - случайное удаление, может терять детали в областях с малой плотностью. Spatial - равномерно прореживает по расстоянию, сохраняет общую форму, но мелкие детали могут сглаживаться. Octree - адаптивно сохраняет структуру, распределяя точки равномерно по пространству

**3. Какой способ оказался самым быстрым?**

Octree (0.181 сек)
